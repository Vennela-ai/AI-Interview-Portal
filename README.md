#  AI Interview Portal

An AI-powered mock interview platform built with **Flask**, **Google Gemini AI**, **SQLite**, and **Bootstrap**. The application helps users prepare for technical interviews by generating AI-based interview questions, evaluating responses, and tracking interview performance.

---

##  Features

*  Secure User Registration and Login
*  User Profile Management
*  Change Password
*  AI-Based Interview Question Generation
*  AI Answer Evaluation using Google Gemini
*  Interview Score and Feedback
*  Interview History
*  Interview Result Storage with SQLite
*  Responsive User Interface using Bootstrap

---

##  Tech Stack

* **Backend:** Flask (Python)
* **Frontend:** HTML, CSS, Bootstrap
* **Database:** SQLite with SQLAlchemy
* **AI Model:** Google Gemini API
* **Authentication:** Werkzeug Password Hashing

##  Installation

1. Clone the repository

```bash
git clone <repository-url>
```

2. Navigate to the project

```bash
cd AI-Interview-Portal
```

3. Install dependencies

```bash
pip install -r requirements.txt
```

4. Run the application

```bash
python app.py
```

5. Open your browser and visit:

```
http://127.0.0.1:5000/
```
